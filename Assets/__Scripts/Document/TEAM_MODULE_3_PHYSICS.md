# 模块 3：棒球物理 & 落地判定 — 分工汇报材料

## 我负责什么

**一句话**：让球从发球点飞到击球区、被球棒打出去、落到地上，然后判断是好球还是界外还是全垒打。

|------|------|--------|
| `Pitcher.cs` | `GamePlay/` | 发球和击球物理：怎么飞、被打后怎么弹、落地上怎么停 |

## 我做了什么

### 1. 微缩世界的发球系统

游戏场景是一个**桌面微缩棒球场**，如果用 Unity 默认的物理重力（9.81），球会以夸张的速度砸下来，和桌面比例完全不搭。

所以自己算了一套微缩物理：
- 自定义重力大小，只有正常重力的 10%（`microGravityScale = 0.1`）
- 关掉 Unity 自带重力，完全接管
- 根据发球点（投手位置）和落点（本垒位置），反算出需要的抛物线初速度
- 公式：水平速度 = 两点距离 ÷ 飞行时间，垂直速度 = 高度差÷时间 − 0.5×g×t

效果：无论球桌缩放多大，球都能精准地从投手飞到击球点，弧度自然。

### 2. 击球物理

当球碰到球棒时（检测 Layer）：

**出球速度**：
- 读取球棒当前的挥动速度（从 `BatCapsuleFollower` 获取）
- 挥得快 → 球飞得远，挥得慢 → 球飞得近
- 有最低辅助速度（轻轻碰到也不会原地掉下去）和最高上限（防止飞出场景）
- 额外加一个向上的速度分量（防止贴地滚太近）

**出球方向**：
- 混合两个方向：球场正前方向 + 真实碰撞反弹方向
- 比重可调（`reflectionDirectionWeight = 0.25`，意思是 25% 反弹 + 75% 朝向球场）
- 防止球被打回身后

**冷却机制**：
- 一次有效击球后 0.3 秒内不再检测球棒碰撞，防止连续碰撞把球打飞

**落地后**：
- 检测到球碰到地面，立刻增大阻力（drag）和旋转阻力，让球自然滚动一段后停下

### 3. 击球判定

球落地后，以本垒为原点判断：

```
                          ╱ 界内扇形区 ╲
                         ╱   45°   45°  ╲
                        ╱                 ╲
          界外 ← ← ←   ╱  有效落地 → 全垒打  ╲   → → → 界外
                      ╱                       ╲
                     ╱___________________________╲
                                本垒
```

- **距离太近**（< 1.5m）：算未击中
- **超出扇形范围**：界外球，0 分
- **扇形内但不够远**（1.5m ~ 6m）：有效落地，按距离算分
- **超过 6m 且落点高度低**：全垒打！分数 × 200

在 Unity Scene 视图中能看到判定区域的 Gizmo 辅助线。

### 4. 10 球循环管理

`ClassicModeRoundManager` 负责整个击球流程：

```
321 倒数 → PitchBall() 发球 → 等待结果
  ├─ 未击中（等 6 秒没人打到） → 0 分
  └─ 击中了 → 球落地 → HitJudge 判定 → 算分 × 连击倍率
→ 显示得分 → 下一球
→ 10 球打完 → 结束，通知结算
```

**连击增益**：
- 连续击中次数越多，得分倍率越高
- 3 连 ×1.2，4 连 ×1.5，5 连 ×1.6
- 一旦没打中，连击归零

**编辑器测试快捷键**：
- 小键盘 1：模拟全垒打
- 小键盘 2：模拟有效落地
- 小键盘 3：模拟界外
- 小键盘 0：模拟未击中
- Enter：开始新一局
- Backspace：中止当前局

## 怎么实现的（核心思路）

**发球**：运动学反推法。知道了起点、终点、飞行时间、重力，反算初速度。用协程驱动物理计算。

**击球**：碰撞检测 → 获取球棒速度 → 映射为出球速度 → 混合方向 → 设置 Rigidbody.velocity。

**判定**：球落地触发的 Collision 取接触点坐标 → 算和本垒的夹角 → 和扇形半角比较 → 算距离 → 匹配阈值 → 得出结果枚举。

**回合循环**：协程状态机，`Idle → Countdown → Pitching → WaitingResult → Scoring → 下一球`，每个阶段等待不同条件。

---

## 遇到的坑

**最大的坑**：击球后球被"秒复位"。球刚被打出去，碰到地面一瞬间触发了两次判定——`OnCollisionEnter` 和 `OnCollisionStay` 各触发一次。第一次判定完进入地面状态（drag 变大），第二次又触发但又说"没有活跃击球"了。后来加了击球后短暂忽略地面的逻辑（`ignoreGroundAfterHitTime = 0.25s`），给球一点飞行时间，等真正落地再处理。
